## ğŸš€ How to Use

### Changing Branding/Content:
1. Open `config.js`
2. Update any value (name, email, products, etc.)
3. Done! No need to touch HTML

### Creating New Pages:
1. Create new HTML file
2. Link the same CSS files:
   ```html
   <link href="colors.css" rel="stylesheet"/>
   <link href="typography.css" rel="stylesheet"/>
   <link href="animations.css" rel="stylesheet"/>
   <script src="config.js"></script>
   ```
3. Use same design system & animations

### Customizing Animations:
- Add animation classes: `animate-fade-in-up`, `animate-bounce`, etc.
- Use delay classes: `animate-delay-1` through `animate-delay-5`
- Hover effects: `hover-lift`, `hover-scale-up`

### Customizing Colors:
- Use variables in CSS: `color: var(--color-primary)`
- Tailwind classes still work: `text-primary`, `bg-primary-dark`
- Add new variables in `colors.css`
